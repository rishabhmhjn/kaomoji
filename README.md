Kaomoji ＼(^o^)／
=================

Random library to generate kaomojis.
To install via npm ([kaomoji](https://npmjs.org/package/kaomoji)),

```
npm install -g kaomoji
```

Usage
-----

### In code

```js
var kaomoji = require('kaomoji');
kaomoji.happy();
kaomoji.sad();
kaomoji.thanks();
kaomoji.congrats();
// please add more :)

kaomoji.happy.all
```

### In command line

```bash
kaomoji [happy|sad|thanks|congrats] [--all|-a]
```
